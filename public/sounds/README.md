# Notification Sounds

Add a `notification.mp3` file to this folder for notification sounds.

## Recommended specifications:
- Format: MP3
- Duration: 0.5-2 seconds
- Size: < 50KB
- Sample: A short, pleasant chime or bell sound

## Free sound resources:
- [Freesound](https://freesound.org/) - Search for "notification" or "chime"
- [Mixkit](https://mixkit.co/free-sound-effects/notification/)
- [Zapsplat](https://www.zapsplat.com/)

## Example sounds to search for:
- "UI notification"
- "App alert"
- "Message received"
- "Bell chime"
